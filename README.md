FUTURE_PE_02 — AI Content Marketing using UGC Ads
📌 Project Overview

This project focuses on creating UGC-style (User Generated Content) ad scripts using AI for a real-world product.

UGC ads are more effective than traditional ads because they feel natural, relatable, and trustworthy.

🎯 Objective
Generate UGC ad scripts using AI
Create multiple hooks for the same product
Follow Problem → Solution → CTA format
Adapt content for different platforms
Produce ready-to-use ad content
🏷️ Product Chosen

GlowPure — D2C Natural Skincare Brand
Target: Women aged 18–35

🛠️ Tools Used
ChatGPT
Claude AI
Google Gemini
✨ Features

✔ Authentic UGC tone
✔ Scroll-stopping hooks
✔ Conversion-focused content
✔ Platform-specific scripts
✔ Reusable prompt system

📱 Ad Scripts

YouTube Shorts
📣 CTAs

📝 Captions
Platform-specific captions

Each output is generated using:

Voice (customer/creator)
